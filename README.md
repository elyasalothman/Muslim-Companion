# رفيق المسلم — نسخة الويب (PWA)

هذه الحزمة جاهزة للرفع على أي استضافة HTTPS. تدعم:
- أذكار صحيحة منظّمة وقابلة للتوسعة (js/adhkar.json)
- مواقيت الصلاة عبر AlAdhan API + عدّ تنازلي + تنبيه
- اتجاه القبلة (مع طلب إذن حساس الاتجاه على iOS)
- مسبحة إلكترونية مع إحصاءات
- وضع التثبيت PWA (manifest + service-worker)

## النشر السريع
1) ارفع محتويات المجلد إلى أي استضافة **HTTPS** (Netlify, Vercel, GitHub Pages مع إعدادات، أو خادم Nginx/Apache).
2) تأكد من أن المسار الجذري يقدّم `index.html` ويمكّن ملفات JSON وJS وservice worker دون تغيير المسارات.
3) افتح العنوان على الهاتف، وستظهر لك إمكانية التثبيت كتطبيق.

## ملاحظات
- الإشعارات والموقع وحساس الاتجاه تتطلب HTTPS وأذونات من المستخدم.
- لتخصيص الأذكار، حرّر `js/adhkar.json`.
- لتغيير ألوان/هوية الواجهة، عدّل `css/styles.css`.
